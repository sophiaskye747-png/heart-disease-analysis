# Heart Disease Analysis

## Overview  
An exploratory data analysis and classification project examining risk factors associated with heart disease.

## Tools Used  
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

## What I Did  
- Cleaned and preprocessed medical data  
- Conducted exploratory data analysis (EDA) with visualizations  
- Built and evaluated a classification model to predict heart disease  
- Interpreted key predictors of risk
